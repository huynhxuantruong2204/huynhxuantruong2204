### 📌 Project Overview
A web-based platform for online document signing, integrating digital signature verification with third-party Certificate Authorities (CAs) to ensure document legality and authenticity. The system includes:
- A **customer-facing site** for users to upload, sign, and manage documents.
- An **admin portal** for monitoring signing activities and managing users.

### ✨ Key Features
- **🔐 Digital Signature Integration**: Validates signatures using third-party APIs (e.g., VNPT-CA, Viettel-CA).
- **⚡ Efficient Document Signing**: Reduces processing time by 80% compared to manual signing.
- **📱 Mobile-Responsive Design**: Works seamlessly on all devices.
- **🔒 Robust Security**: Implements SSL/TLS encryption, JWT authentication, and RBAC.
- **🤝 Multi-Signature Support**: Allows multiple stakeholders to sign a single document.
